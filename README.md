## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# deploy on lambda localhost
$ serverless offline start

# deploy on lambda production
$ npm run server:deploy
```
